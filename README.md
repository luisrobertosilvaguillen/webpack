# Webpack Examples

## About
Basics Webpack examples 


## Intructions
1.

  - `cd webpack`
 
 2.
 
  - `npm install`
 
 3.
 
  - Look for an example and execute the script. Example: `npm run build:video`
  
 4.
 
  - Find the folder ./dist inside the script folder executed previously
